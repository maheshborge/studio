# **App Name**: MaziLink

## Core Features:

- Profile Creation: Users can create and manage their profiles with basic information.
- Content Aggregation: The app will parse content from www.mazisheti.org and display them, including videos and other content.
- AI-Powered Content Summarization: Use AI to generate brief summaries of the aggregated content, which the app can use as a tool to increase user engagement.
- User Authentication: Secure user login and registration functionality.
- Search Functionality: Enable users to search for specific content within the app.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and authority.
- Background color: Very light blue (#E8EAF6) to keep the app approachable and avoid visual fatigue.
- Accent color: Vivid purple (#9C27B0) for highlights and interactive elements.
- Body and headline font: 'Inter', a neutral sans-serif font.
- Use clean, modern icons to represent various content categories.
- A card-based layout for displaying content snippets. Content from source website (mazisheti.org) should render correctly and be responsive.
- Subtle transitions and animations to improve user experience when navigating and loading content.
