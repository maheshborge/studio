# MaziSheti - MIDAS Platform

हे **MIDAS by MaziSheti** प्लॅटफॉर्मचे अधिकृत वेब अ‍ॅप आहे.

## महत्त्वाच्या सूचना (Deployment)

१. **index.html डिलीट करा:** जर तुमच्या रूट फोल्डरमध्ये `index.html` असेल तर ती डिलीट करा. Next.js मध्ये तिची गरज नसते.
२. **कोड पुश करा:**
```bash
git add .
git commit -m "Fix build detection by removing static files and updating engine"
git push origin main
```

३. **Firebase Console:** कोड पुश केल्यावर 'Release in progress' कडे लक्ष द्या. आता ते यशस्वी व्हायला हवे.
