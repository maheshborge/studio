# MaziSheti - MIDAS Platform

हे **MIDAS by MaziSheti** प्लॅटफॉर्मचे अधिकृत वेब अ‍ॅप आहे.

## कोड तुमच्या PC वर कसा घ्यायचा? (How to get code on your PC)

फायरबेस स्टुडिओमध्ये 'Direct Download' बटण नसल्यामुळे, तुम्ही खालील दोन मार्ग वापरू शकता:

### मार्ग १: GitHub वरून क्लोन करणे (शिफारस केलेले - Recommended)
१. फायरबेस स्टुडिओमधील खालच्या बाजूला असलेल्या **Terminal** मध्ये खालील कमांड्स एकेक करून टाईप करा आणि 'Enter' दाबा:
   - `git add .`
   - `git commit -m "Update project files"`
   - `git push origin main`
२. आता तुमच्या पीसीवर (PC) तुमच्या आवडीचे फोल्डर उघडा आणि तिथे कमांड प्रॉम्ट (CMD) मध्ये खालील कमांड टाकून कोड डाऊनलोड करा:
   - `git clone https://github.com/maheshborge/firebase-midas.git`

### मार्ग २: मॅन्युअली कॉपी-पेस्ट करणे
१. फायरबेस स्टुडिओच्या डाव्या बाजूला असलेल्या फाईल एक्सप्लोररमध्ये हवी ती फाईल उघडा.
२. त्यातील सर्व कोड कॉपी (Ctrl+C) करा.
३. तुमच्या पीसीवर नवीन फाईल बनवून तिथे तो पेस्ट (Ctrl+V) करा. 
४. लक्षात ठेवा: फाईलचे नाव आणि फोल्डर स्ट्रक्चर (उदा. `src/app/page.tsx`) स्टुडिओप्रमाणेच असावे.

## मॅन्युअल अपलोड (Manual GitHub Upload) करताना महत्त्वाच्या सूचना:
जर तुम्ही GitHub च्या वेबसाईटवरून फाईल्स अपलोड करत असाल, तर:
- फक्त `src`, `public` फोल्डर्स आणि `package.json`, `apphosting.yaml`, `next.config.ts`, `tailwind.config.ts`, `tsconfig.json` या फाईल्स निवडा.
- **index.html अपलोड करू नका.** Next.js अ‍ॅपमध्ये ही फाईल असल्यास 'Build Failure' येतो.
