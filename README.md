
# MaziSheti - MIDAS Platform (Clean Reboot)

हा प्रकल्प अगदी सुरुवातीपासून (Scratch) पुन्हा उभा केला आहे. 

## १. प्रोजेक्ट सिंक कसा करावा?
टर्मिनलमध्ये खालील कमांड्स एकामागून एक टाका:
- `git add .`
- `git commit -m "Clean reboot: Fresh start"`
- `git push origin main`

## २. मॅन्युअल अपलोडसाठी टीप
जर तुम्ही GitHub वर फाईल्स ड्रॅग-अँड-ड्रॉप करत असाल, तर खात्री करा की:
- **`index.html` फाईल कधीही अपलोड करू नका.**
- `src` फोल्डर आणि `package.json` यांसारख्या फाईल्स मुख्य फोल्डरमध्ये आहेत.
